2-2 스크롤 트리거 & 핀
js
Copy
Edit
gsap.registerPlugin(ScrollTrigger);

// 글자 영역의 실제 위치·크기 계산
const box = document.querySelector('.hero__videoWrap')
                 .getBoundingClientRect();
const vw  = window.innerWidth,
      vh  = window.innerHeight;
const scaleX = vw / box.width,
      scaleY = vh / box.height,
      scale  = Math.max(scaleX, scaleY);

gsap.timeline({
  scrollTrigger:{
    trigger:".hero",
    start:"top center",
    end:"+=200%",
    scrub:true,
    pin:true
  }
})
.fromTo(".hero__videoWrap",
  {scale:1, x:0, y:0},
  {scale, x:-box.left, y:-box.top, ease:"power1.out"}
);
scrub:true → 스크롤 위치 = 애니메이션 진행도

pin:true 로 영상 고정 후 다음 섹션 넘어가도 자연스럽게 해제

4. 대체(바닐라) 접근
Intersection Observer로 트리거 감지 → position:fixed 클래스 토글, CSS transition:transform 0.8s cubic-bezier… 이용. 성능은 좋지만 스크롤-구간 비례 제어(스크럽)가 어렵습니다.

Lenis / Smooth-Scroll + CSS sticky 조합도 가능하지만, 고정(pin)·스크럽을 동시에 쓰려면 결국 GSAP ScrollTrigger가 가장 쉬운 선택입니다.
